Projekat koristi python jezik i kreiran je u flask okruženju. Za pokretanja projekta je potrebno locirati folder, kreirati i aktivirati virtualno okruženje, instalirati potrebno pakete "pip install -r requirements.txt" i unijetu komandu "python app.py".

Za provjeru pravopisa i ispravku pravopisa se koristi biblioteka symspellpy. 
Symspellpy je python implementacija SymSpell algoritma za ispravku pravopisa koja unaprijed izračunava sve moguće varijante za dati riječnik unutar određene udaljenosti. 

Za provjeru i ispravku gramatike koristi se language_tool_python, Python wrapper za LanguageTool.
